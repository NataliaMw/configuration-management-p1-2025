GENERAL OBJECTIVE OF THE SUBJECT

Develop a software system using a development methodology and control tools to obtain a properly tested and versioned final product, with readable and easily maintainable code.

SUBJECT DESCRIPTION

The professional training course addresses the development of a medium-complexity software system using good programming practices, test cases, version control tools, code analysis, and automatic test generation. Additionally, it develops non-technical skills such as: teamwork applying a software development methodology, continuous learning through research of programming languages and/or tools required by the client, and oral and written communication.

PRIOR KNOWLEDGE AND/OR COMPETENCIES

Programming language and software design tools.

LEARNING OUTCOMES OF THE SUBJECT

  1. Use a source code control tool for the management and versioning of a moderately complex software system developed in a team.
  2. Use a coding standard to ensure code quality in a given project.
  3. Develop a set of test cases for the correct operation of a software system.
  4. Analyze the impact of a change request to an existing software product determining its feasibility.
